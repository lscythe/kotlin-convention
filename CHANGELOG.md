## [1.0.0] - 2026-01-09

### Added

- Initial release
- Config module with CommonSettingsPlugin
- Versioning module with tag-based versioning
- Quality module with ktlint, detekt, spotless integration
- JVM module with KotlinLibraryPlugin
- Android module with AndroidLibraryPlugin and AndroidApplicationPlugin
- Multiplatform module with MultiplatformLibraryPlugin
- Testing module with TestProject utilities
- GitHub workflows for build and publish
- Release script with semantic versioning
